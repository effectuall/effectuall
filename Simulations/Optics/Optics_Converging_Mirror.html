    Image Formation with converging mirror

    Description

    For simulation of image formation in convex mirror. Use the control menu, to change the lens focus, to move the object and adjust the object height.

    Observe that the focal length is positive here.

    For visualising and understanding the ray diagram of the convex mirror better, you can select pre-set object distance for the lens with focal length 5 cm.

    Lens Formula

    $$\frac{1}{f} = \frac{1}{v}+\frac{1}{u} $$ \[\text{or}\] $$\frac{1}{v} = \frac{1}{f}-\frac{1}{u} $$

        ➤ Focal length of convex mirror (converging mirror) are positive.
        ➤

    Magnification

    $$m=-\frac{v}{u} $$
